# MenúYa - Sistema de pedidos

## Integrantes
- Juan Ramón Espinal Coto  
- Walter Alexander Ramírez Benítez

## Problema
Los pequeños negocios no tienen sistemas digitales para gestionar pedidos de forma eficiente.

## Solución
Se desarrolló una aplicación web con Vue.js que permite:
- Seleccionar categoría de comida
- Ver opciones dinámicas
- Agregar pedidos
- Calcular totales automáticamente
- Eliminar pedidos
- Validar datos

## Tecnologías
- HTML
- CSS
- Vue.js

## Preguntas

### ¿Qué es Vue.js?
Framework para crear interfaces dinámicas.

### Variables reactivas
nombreCliente, categoria, comidaSeleccionada, cantidad, pedidos, error.

### v-bind vs v-model
v-bind enlaza atributos, v-model enlaza inputs con datos.

### Evento
@click para agregar y eliminar pedidos.

### v-for
Para listar pedidos y comidas.

### v-if
Para mostrar errores y total.

### Validación
Se valida que no haya campos vacíos y cantidades inválidas.
